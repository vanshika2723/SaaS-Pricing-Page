# SaaS Pricing Page UI

## 🚀 Overview
This project is a responsive SaaS pricing page built using HTML, CSS, and JavaScript. It includes pricing tiers, feature comparison, and interactive UI elements.

---

## 🛠 Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (Vanilla)

---

## ✨ Features
- Responsive mobile-first design
- Monthly / Yearly pricing toggle
- Highlighted recommended plan
- Feature comparison table
- FAQ section
- Dark mode toggle
- Hover micro-interactions

---

## ⚡ Approach
I started by designing a clean layout focusing on SaaS conversion principles. Then I structured the pricing cards, followed by interactivity using JavaScript. Finally, I added responsiveness and dark mode for better UX.

---

## 🧠 Challenges Faced
- Managing pricing toggle logic
- Ensuring responsive layout across devices
- Keeping UI minimal yet attractive
